# SoTrail  — AI-Powered Travel Planning Platform

> **Plan smarter. Travel better.**  
> SoTrail is a full-stack travel planning application that combines real-time Amadeus flight & hotel data with Groq's Llama 3.3 70B AI for intelligent trip planning, personalized itinerary generation, and a RAG-powered travel chatbot.

##  Features

| Feature | Description |
|---------|-------------|
|  **Flight Search** | Real-time flights via Amadeus API with pricing, stops, duration |
|  **Hotel Search** | Live hotel availability with estimated and real Amadeus prices |
|  **Train Options** | Indian Railways train suggestions with class types and booking links |
|  **Bus Search** | Multi-operator bus options with RedBus deep links |
|  **Cab Estimation** | Uber cab price estimates for short-to-medium distances |
|  **AI Chatbot** | Context-aware travel assistant powered by Llama 3.3 70B via Groq |
|  **AI Itinerary** | Personalized day-by-day itineraries generated by LLM |
|  **Route Map** | Interactive map showing origin to destination route |
|  **Analytics** | Search history and popular destination tracking via MongoDB/ChromaDB |
|  **RAG Search** | ChromaDB vector database for semantic destination guide retrieval |

##  Getting Started

### Prerequisites

- **Python** 3.12+
- **Node.js** 20+
- **API Keys** (see below)

### Obtaining API Keys

| API | Where to Get | Free Tier |
|-----|-------------|-----------|
| Amadeus API Key + Secret | [developers.amadeus.com](https://developers.amadeus.com/) | ✅ 2,000 requests/month |
| Groq API Key | [console.groq.com](https://console.groq.com/) | ✅ Generous free tier |

---

###  Quick Start

**1. Clone the repository**
```bash
git clone https://github.com/broskifoo/SoTrail.git
cd SoTrail
```

**2. Backend setup**
```bash
cd backend

# Create virtual environment
python -m venv venv

# Activate (Windows)
venv\Scripts\activate
# Activate (macOS/Linux)
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Configure environment
cp .env.example .env
# Edit .env and add your API keys
```

**3. Configure `.env`**
```env
AMADEUS_API_KEY=your_amadeus_api_key
AMADEUS_API_SECRET=your_amadeus_api_secret
GROQ_API_KEY=your_groq_api_key
```

**4. Start the backend**
```bash
python main.py
# Backend running at: http://localhost:8000
# API Docs at: http://localhost:8000/docs
```

**5. Frontend setup** (in a new terminal)
```bash
cd frontend
npm install
cp .env.example.txt .env.local
# Edit .env.local: NEXT_PUBLIC_API_URL=http://localhost:8000
npm run dev
# Frontend running at: http://localhost:3000
```

##  API Reference

### Base URL
```
http://localhost:8000
```

### Endpoints

#### `POST /api/search-all`
Search for all transport and accommodation options in one request.

**Request body:**
```json
{
  "origin": "Mumbai, Maharashtra",
  "destination": "Goa, Goa",
  "departureDate": "2025-04-15",
  "departureTime": "09:00",
  "returnDate": "2025-04-20",
  "returnTime": "18:00",
  "travelers": 2,
  "tripType": "domestic"
}
```

**Response:** Flights, hotels, trains, buses, cabs, route coordinates, and summary.

---

#### `POST /api/chat`
AI-powered travel assistant with trip context.

**Request body:**
```json
{
  "message": "What are the best beaches in Goa?",
  "context": {
    "origin": "Mumbai",
    "destination": "Goa"
  }
}
```

---

#### `POST /api/generate-itinerary`
Generate a personalized day-by-day itinerary using Llama 3.3.

**Request body:**
```json
{
  "origin": "Delhi",
  "destination": "Rajasthan",
  "departureDate": "2025-05-01",
  "returnDate": "2025-05-07",
  "travelers": 2,
  "selectedFlight": {},
  "selectedHotel": {}
}
```

---

#### `GET /api/analytics`
Get search analytics — recent searches and popular destinations.

##  How It Works

### Multi-Modal Transport Search
```
User Input → Airport Code Lookup (airportsdata + geopy)
          → Amadeus API (Flights + Hotels)
          → Indian Railways simulation (domestic trains)
          → RedBus integration (domestic buses)
          → Uber price estimation (cab)
          → Combined response
```

### RAG-Powered Chatbot
```
User Message → Embedding (Sentence Transformers)
             → Vector Search (ChromaDB)
             → Retrieved destination context
             → Llama 3.3 70B (Groq) with context
             → Contextual travel advice
```

### AI Itinerary Generation
```
Trip dates + destination → Groq LLM prompt
                         → Structured JSON itinerary
                         → Day-by-day activities
                         → Meal + timing suggestions
```
